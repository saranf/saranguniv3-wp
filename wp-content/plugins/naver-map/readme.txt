Naver Map 플러그인은 네이버 지도 API를 이용하여 워드프레스에서 쉽게 사용할 수 있도록 도와줍니다.

== Description ==

This plugin uses Naver API and helps users easy to use in WordPress.

본 플러그인은 네이버 지도API를 이용하는 관계로 <a href="http://dev.naver.com/openapi/apis/map/javascript_2_0/example" target="_blank">네이버 지도API 키</a>를 발급받아야 사용가능한 플러그인입니다.

* 플러그인 옵션에서 주소 검색을 통해 지도 숏코드 생성합니다.
* 포스트/페이지 편집모드에서 지도 숏코드를 추가할 수 있도록 개선할 예정입니다.
* 다국어 지원 : 한국어, 영어

== Installation ==

플러그인을 설치하시기 위해서 다음과 같은 순서로 진행하시면 됩니다.

1. 네이버 -> 개발자 센터 -> 오픈API -> 지도API -> 키 등록/관리 화면에서 지도API 키를 발급 받습니다.
2. Naver Map 플러그인을 다운로드 받습니다.
3. 다운로드 받은 플러그인을 FTP를 통해 Plugins 폴더로 업로드 합니다.
4. 플러그인 관리에서 Naver Map 플러그인을 활성화 시킵니다.
5. 설정 > Naver Map로 접속합니다.
6. 네이버 지도API 키를 등록합니다.(키 등록을 해야만 주소검색이 가능합니다.)
7. 옵션을 설정하고 지도 숏코드를 복사합니다.
9. 복사한 숏코드를 포스트나 페이지에 붙여넣습니다.
